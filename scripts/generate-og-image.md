# Generar Imagen Open Graph

Para crear la imagen Open Graph (`/public/images/og-image.jpg`), necesitas una imagen de **1200x630 píxeles**.

## Opciones:

### Opción 1: Usar una herramienta online
- [Canva](https://www.canva.com/) - Tiene plantillas para Open Graph
- [Bannerbear](https://www.bannerbear.com/)
- [OG Image Generator](https://www.opengraph.xyz/)

### Opción 2: Crear manualmente
1. Abre un editor de imágenes (Photoshop, GIMP, etc.)
2. Crea un canvas de 1200x630 píxeles
3. Incluye:
   - Logo de Nortesur Travel
   - Texto: "Nortesur Travel - Tu agencia de viajes de confianza"
   - Colores de la marca (#033671, #6D4C05)
   - Imagen de fondo relacionada con viajes (opcional)

### Opción 3: Usar una imagen existente
Puedes usar una de las imágenes del hero (`head1.jpg`, `head2.jpg`, etc.) y redimensionarla a 1200x630px.

## Ubicación del archivo:
```
/public/images/og-image.jpg
```

## Nota:
Mientras no tengas la imagen, el sitio funcionará correctamente, pero las redes sociales mostrarán una imagen por defecto o ninguna imagen al compartir el sitio.
